This game was made to introduce MITRE ATT&CK to cyber security students.
To give them the chance to rehearse their knowledge in a fun way.
This game consists of several questions with multiple answers to choose from.
Each question can have more than one correct answer, the player can choose any.

Each question in this game is about an attack technique for an adversary group, and can have more than one correct answer,
the player can choose any correct mitigation to stop the attack.


Each question in this game is about an attack technique for an adversary group, and can have more than one correct answer, the player can choose any correct mitigation to stop the attack. After answering each question, instant feedback will be given to the player.


ATT&CK To Defend gives the player the opportunity to visit the ATT&CK matrix for more information, by offering links related to each attack.
